## 快速开始

### 第一步：添加配置

1. 打开Clash Mi应用
2. 选择APP首页菜单中的"我的配置"
3. 点击右上角的 ➕ 号按钮
4. 选择"添加配置链接"
5. 输入（粘贴）您的机场订阅链接
6. 点击右上角✅确认完成添加
![添加配置链接](/assets/clashmi/clashmi2.png "添加配置链接")

### 第二步：启动连接

1. 返回APP首页
2. 滑动右上角打开连接按钮 ▶️
3. 首次使用会提示添加VPN配置，点击允许,输入密码
4. 连接成功后即可开始使用
![连接成功](/assets/clashmi/clashmi7.png "连接成功")

## 详细配置说明

### 订阅管理

**添加订阅的几种方式**:

1. **URL链接**: 直接输入机场提供的订阅链接
2. **二维码扫描**: 扫描机场提供的订阅二维码
3. **本地文件**: 导入本地的yaml配置文件
4. **剪贴板导入**: 从剪贴板粘贴配置内容

**订阅更新**:
- 支持手动更新和自动更新
- 可设置更新间隔时间
- 更新时会保留用户自定义设置

### 代理模式选择

Clash Mi支持多种代理模式：

1. **规则模式**: 根据规则自动选择直连或代理（推荐）
2. **全局模式**: 所有流量通过代理
3. **直连模式**: 所有流量直接连接

### 节点选择

**手动选择**:
- 在节点列表中点击选择具体节点
- 支持按地区、类型筛选节点
- 显示节点延迟和负载信息

![节点选择](/assets/clashmi/clashmi9.png "节点选择")
![节点选择](/assets/clashmi/clashmi10.png "节点选择")

**自动选择**:
- 支持负载均衡
- 支持故障转移
- 支持延迟最低自动选择

### 自定义规则

**支持的规则类型**:
- DOMAIN: 域名匹配
- DOMAIN-SUFFIX: 域名后缀匹配
- DOMAIN-KEYWORD: 域名关键词匹配
- IP-CIDR: IP地址段匹配
- GEOIP: 地理位置IP匹配

**规则优先级**:
规则按照配置文件中的顺序执行，越靠前的规则优先级越高。

### 分流配置

**常见分流场景**:
- 国内网站直连
- 国外网站代理
- 流媒体专用线路
- 游戏加速线路
- 广告拦截

## YAML配置参考

对于高级用户，Clash Mi支持完整的YAML配置文件。以下是基本配置结构：

```yaml
# 基础配置
port: 7890
socks-port: 7891
allow-lan: false
mode: rule
log-level: info

# 代理服务器配置
proxies:
  - name: "节点名称"
    type: ss
    server: server.example.com
    port: 443
    cipher: aes-256-gcm
    password: "password"

# 代理组配置
proxy-groups:
  - name: "PROXY"
    type: select
    proxies:
      - "节点名称"
      - DIRECT

# 规则配置
rules:
  - DOMAIN-SUFFIX,google.com,PROXY
  - DOMAIN-KEYWORD,github,PROXY
  - GEOIP,CN,DIRECT
  - MATCH,PROXY
```

## 常见问题

## 安装相关问题

### Q: iOS版本无法下载或安装

**A:** 解决方法：
1. **检查Apple ID地区**: 确保使用非中国大陆地区的Apple ID
2. **网络问题**: 确保网络连接正常
3. **存储空间**: 确保设备有足够的存储空间
4. **系统版本**: 确认iOS版本在15.0以上

### Q: Android版本安装失败

**A:** 常见解决方案：
1. **开启未知来源**: 在设置中允许安装未知来源应用
2. **小米用户**: 按照前面提到的特殊步骤操作
3. **存储权限**: 确保应用有存储权限
4. **安装包完整性**: 重新下载安装包

## 连接相关问题

### Q: 添加订阅后无法获取节点

**A:** 排查步骤：
1. **订阅链接检查**: 确认订阅链接正确且有效
2. **网络连接**: 确保设备能够访问订阅服务器
3. **订阅格式**: 确认订阅为Clash兼容格式
4. **防火墙设置**: 检查是否被防火墙阻止

### Q: 连接成功但无法上网

**A:** 解决方法：
1. **检查代理模式**: 确认选择了正确的代理模式
2. **节点状态**: 尝试切换其他节点
3. **DNS设置**: 检查DNS配置是否正确
4. **系统代理**: 确认系统代理设置正确

### Q: 连接速度慢或不稳定

**A:** 优化建议：
1. **节点选择**: 选择延迟较低的节点
2. **线路类型**: 尝试不同类型的线路（如IPLC、IEPL）
3. **时间段**: 避开网络高峰期使用
4. **配置优化**: 调整连接参数

## 使用相关问题

### Q: 如何备份和恢复配置

**A:** 备份方法：
1. **导出配置**: 在设置中导出当前配置
2. **云端同步**: 使用iCloud或其他云服务同步
3. **手动备份**: 记录重要的订阅链接和设置

### Q: 应用闪退或无响应

**A:** 解决步骤：
1. **重启应用**: 完全关闭后重新打开
2. **重启设备**: 重启手机或电脑
3. **清除缓存**: 清除应用缓存数据
4. **重新安装**: 卸载后重新安装应用

### Q: 如何更新应用

**A:** 更新方式：
- **iOS**: 通过App Store或TestFlight更新
- **Android**: 下载新版APK覆盖安装
- **macOS**: 下载新版本替换旧版本

## 高级问题

### Q: 自定义规则不生效

**A:** 检查要点：
1. **规则语法**: 确认规则语法正确
2. **规则顺序**: 检查规则优先级
3. **配置重载**: 修改后需要重新加载配置
4. **日志查看**: 通过日志查看规则匹配情况

### Q: 特定应用无法使用代理

**A:** 解决方案：
1. **应用权限**: 确保应用有网络权限
2. **分应用代理**: 检查分应用代理设置
3. **规则配置**: 为特定应用配置专门规则
4. **兼容性**: 某些应用可能不兼容代理
