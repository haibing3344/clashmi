## 基本问题

### ClashMi是什么？

ClashMi是一款基于Mihomo(Clash Meta)内核的移动端代理工具，完全免费且开源。作为专为移动设备优化的代理客户端，ClashMi在保持强大功能的同时，更加注重移动端的用户体验和操作便捷性。

### ClashMi支持哪些平台？

ClashMi支持以下平台：
- iOS: iPhone/iPad (iOS 15+)
- Windows: Windows 10/11 (Windows 10专业版或以上)
- Android: 安卓手机/平板 (Android 8+)
- HarmonyOS: 华为鸿蒙系统

### ClashMi是否收费？

ClashMi是完全免费且开源的软件，您可以免费使用所有功能，无需付费。ClashMi的开发团队致力于提供高质量的免费代理工具，让更多用户能够便捷地使用互联网。

### 什么是机场？如何获取订阅链接？

机场是提供代理服务的服务商，通常以订阅的形式提供多个代理节点。要获取订阅链接，您需要：

1. 选择一个机场服务并注册账号
2. 购买适合您需求的套餐
3. 在机场网站的用户中心或仪表盘找到「订阅链接」或「Clash配置」
4. 复制该链接并在ClashMi中添加

您可以在我们的[机场推荐](/airport)页面找到一些推荐的机场服务。

## 使用问题

### 如何添加订阅？

添加订阅的步骤如下：

1. 打开ClashMi应用
2. 选择APP首页菜单中的"我的配置"
3. 点击右上角的 ➕ 号按钮
4. 选择"添加配置链接"
5. 输入（粘贴）您的机场订阅链接
6. 点击右上角✅确认完成添加

详细步骤请参考我们的[使用教程](/tutorial)。

### 如何选择最佳节点？

选择最佳节点的方法：

- 使用延迟测试功能，选择延迟最低的节点
- 根据您的使用需求选择不同地区的节点（如观看特定地区的流媒体内容）
- 尝试使用自动选择功能，如负载均衡或故障转移
- 高峰时段可能需要尝试不同节点以获得最佳性能

### 不同代理模式有什么区别？

ClashMi支持以下代理模式：

- **规则模式：** 根据预设规则自动选择直连或代理。这是推荐的模式，可以智能地决定哪些流量需要代理，哪些直接连接，提高效率和速度。
- **全局模式：** 所有流量都通过代理。适用于需要全部流量代理的场景，但可能会影响一些本地服务的访问速度。
- **直连模式：** 所有流量直接连接，不使用代理。当您暂时不需要代理服务时可以使用此模式。

## 故障排除

### 无法连接到服务器怎么办？

如果无法连接到服务器，请尝试以下解决方法：

- 检查您的订阅是否过期，如果过期请续费
- 尝试更新订阅，获取最新的节点信息
- 检查您选择的节点是否可用，尝试切换到其他节点
- 检查您的网络连接是否正常
- 尝试重启应用或设备
- 如果使用的是移动网络，尝试切换到Wi-Fi，或反之

### 连接速度慢怎么解决？

提高连接速度的方法：

- 使用延迟测试功能选择延迟最低的节点
- 优先选择IEPL/IPLC等专线节点，这些通常提供更好的速度和稳定性
- 在高峰时段，尝试切换到不同地区的节点
- 检查您的本地网络状况，如Wi-Fi信号强度
- 尝试使用规则模式而非全局模式，减少不必要的代理流量
- 关闭设备上其他占用带宽的应用

### 应用闪退或无响应怎么办？

解决应用闪退或无响应的方法：

- 确保您使用的是最新版本的ClashMi，如果不是，请更新到最新版本
- 尝试强制关闭应用后重新启动
- 重启您的设备
- 检查设备存储空间是否充足
- 如果问题持续，尝试卸载并重新安装应用
- 卸载前，可以先备份您的配置

## 高级问题

### 如何自定义规则？

自定义规则的步骤：

1. 进入设置页面
2. 选择"规则设置"
3. 点击"添加规则"
4. 设置规则类型（如域名、IP、端口等）
5. 输入匹配内容
6. 选择代理策略（如直连、代理、拒绝等）
7. 保存规则

自定义规则可以帮助您更精细地控制网络流量的路由方式，满足特定的使用需求。

### 如何备份和恢复配置？

备份配置的步骤：

1. 进入设置页面
2. 选择"备份与恢复"
3. 点击"创建备份"
4. 选择备份保存位置

恢复配置的步骤：

1. 进入设置页面
2. 选择"备份与恢复"
3. 点击"从备份恢复"
4. 选择备份文件
5. 确认恢复

定期备份配置可以帮助您在更换设备或重装应用后快速恢复之前的设置。
